     │    - OU utiliser un jeton Nature :  │

     │      - Remplacer les jetons Faune   │
     │ 3. Placer la tuile Habitat          │
     │    - Adjacente à une autre tuile    │

     │    - Alignement des terrains conseillé│
     │ 4. Placer le jeton Faune            │
     │    - Sur une tuile compatible       │
     │    - Sinon, replacé dans le sac     │
     │ 5. Gagner un jeton Nature (si sur Keystone) │
     │ 6. Remplacer la tuile et le jeton Faune     │
